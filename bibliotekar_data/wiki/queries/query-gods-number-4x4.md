# Query: God’s Number 4x4

Exact God’s number for the 4x4x4 Rubik’s Cube is unknown. Current best-known bounds place the diameter between 35 and 55 moves in outer block turn metric, 32–53 moves in single slice turn metric, and 29–53 moves in block turn metric. Probabilistic estimates suggest 48 moves in quarter-turn metric and 41 moves in half-turn metric.
