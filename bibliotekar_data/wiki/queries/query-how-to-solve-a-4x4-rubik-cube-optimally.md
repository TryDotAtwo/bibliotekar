# Query: How to solve a 4x4 Rubik cube optimally?

# Source: rubiks_4x4_summary.txt

Solving a 4×4×4 Rubik's cube commonly uses the reduction method. First solve all six centres, then pair each set of two wing edge pieces to form twelve dedges. Once centres and dedges are solved, turn only outer layers to solve the cube like a 3×3×3.
